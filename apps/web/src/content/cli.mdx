# CLI

Util UI components can be added using the shadcn cli :

```sh
npx shadcn@latest add https://utilui.com/r/button.tsx
```

You can read more from shadcn here : [ui.shadcn.com/docs/cli](https://ui.shadcn.com/docs/cli)
